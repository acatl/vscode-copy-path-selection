# Changelog

All notable changes to **Copy Path and Selection** will be documented here.

## [0.0.2] - 2026-03-01

### Fixed

- Command ID mismatch that caused "command not found" when installed from VSIX.

## [0.0.1] - 2024-01-01

### Added

- Initial release.
- Copy file path + cursor position or selection range to clipboard.
- Compact format: `path:line`, `path:line:col-col`, `path:line:col-line:col`.
- Status bar confirmation message on copy.
- Keybinding: `Ctrl+Alt+Cmd+C` (Mac) / `Ctrl+Alt+Shift+C` (Windows/Linux).
